# 全视者：社交媒体领域指导

在社交媒体领域进行模拟时，你应特别关注以下方面：

## 初始化场景时

- 根据内容类型和发布者特征，判断最可能被触达的初始用户群体
- 考虑发布时间对初始曝光的影响
- 评估内容与平台调性的匹配度

## 裁决涟漪传播时

- 平台的推荐算法如何影响内容可见性
- 早期互动对后续曝光的放大效应
- 不同用户群体之间的圈层壁垒强度
- 内容的情绪唤起程度对传播速度的影响
- 是否有竞争性内容分流注意力
- 内容发布后，平台算法必然推送给一部分初始受众。因此首轮传播中，普通用户群体（Sea）一定会被触达
- 大多数社交媒体内容的传播曲线是"快速冲高 + 缓慢衰减"，能持续多日增长的内容是极少数
- 社交媒体上 90%+ 的内容互动量低于创作者的期望。保持审慎预期，避免过度乐观

## 检测涌现时

- 多个不相关群体对同一内容产生共鸣（破圈）
- 内容语义发生未预期的漂移（模因变异）
- 群体反应强度超出内容本身"应有"的影响力
- 出现自组织的话题标签或二次创作浪潮
- 情绪从一致转向极化（舆论分裂）

## 判断相变时

- 传播速度的不连续跳跃（而非平滑增长）
- 参与群体构成的突然变化
- 主流情绪在短时间内翻转
- 从"讨论"模式切换到"站队"模式
