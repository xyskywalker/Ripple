# 时尚/鞋服零售行业画像

> 本画像为 PMF 验证的行业认知层，在 INIT 阶段注入 OmniscientAgent，校准整个模拟的行业基线。
> 时尚零售是典型的"趋势驱动、库存杀手"赛道，PMF 验证必须区分一次性爆款热度与可持续品牌力。

## 1. 行业概述

中国服装市场规模约 2.6 万亿元，是一个高度季节性驱动、趋势快速轮转的行业。市场格局呈"哑铃型"——头部国际品牌（Nike、Zara、优衣库）和大量中小品牌/白牌并存，中间层品牌生存空间被持续挤压。品牌 3 年存活率不足 30%，大量新品牌在第二年库存积压中死亡。

核心商业逻辑是**"趋势判断 × 供应链速度 × 库存周转"**。时尚零售的利润不在于卖得多贵，而在于卖得多快——库存周转天数超过 90 天就进入危险区域，超过 120 天基本宣告失败。每一季未售出的库存都在以每月 10%-15% 的速度贬值。

线上渠道占比已超过 50%，但线下体验（试穿、触感、即时获得感）仍不可替代。社交媒体——特别是小红书和抖音——已成为核心种草渠道，"被看到穿"比"穿着舒服"在很多品类中更重要。这意味着时尚零售的 PMF 不仅是功能性满足，更是社交价值的交付。

库存是最大的杀手。无数品牌不是因为"卖不出去"而死，而是因为"卖得不够快"而被库存拖死。这在 PMF 验证中意味着：即使产品有需求，如果需求无法在季节窗口内集中爆发，仍然可能是致命的。

## 2. 用户决策模型

### 认知阶段

时尚消费的认知路径高度社交化和视觉化：
- **社交媒体种草**（小红书 OOTD、抖音穿搭视频）：年轻消费者的第一信息源，占认知来源的 40%+
- **明星/KOL 同款效应**：看到博主或明星穿着后产生"我也要"的欲望
- **朋友/同事的穿着**：线下社交场景中的"活广告"，"你这件衣服哪里买的"是高频触发场景
- **逛街偶遇**（商场/买手店）：线下空间的视觉冲击和触感体验仍然强大
- **电商平台推荐流**：基于浏览历史的算法推荐，尤其在换季节点

### 评估阶段

时尚品的评估是感性与理性的混合：
- **视觉判断**（3 秒决定）：款式是否符合个人审美、是否"好看"——这是第一道也是最重要的筛选
- **价格锚定**：与心理预期价格带对比（"这个品牌/品质应该值多少钱"）
- **品牌认知**：是否是认可的品牌、品牌调性是否与自我形象匹配
- **社交验证**：有没有人穿过、穿出来的效果怎么样、会不会被"吐槽"
- **实穿性评估**：尺码是否合适、面料是否舒服、能搭配几套衣服（理性回归）

### 决策阶段

时尚消费的决策周期因价格带差异巨大：
- **快时尚（< 300 元）**：决策周期数分钟，冲动消费占比高，"买错了也不心疼"
- **轻奢/中端（300-2000 元）**：决策周期数天，会比较多个品牌和款式，关注性价比
- **高端/设计师品牌（> 2000 元）**：决策周期可达数周，需要多次接触（看到→种草→试穿→购买）

共同特征是**高度情感驱动**——"我穿上它的感觉"往往比功能性比较更有决定力。

### 复购路径

时尚零售的复购逻辑与快消品截然不同——消费者很少买同一件衣服两次，复购是对**品牌**的复购：
- **品牌忠诚复购**（最强信号）：认准某个品牌的风格/品质/版型，新品上市时主动关注和购买
- **季节性回访**：每到换季节点会回到上一季满意的品牌查看新品
- **触发式复购**：看到品牌的新一季推广或朋友穿着后回购
- **无品牌复购**：按需购买，不关注品牌，哪个便宜/好看买哪个——这不是品牌层面的 PMF 信号

## 3. PMF 信号矩阵

### 强 PMF 信号（行为层面）

- **正价购买+不退货**：消费者在非促销场景下按原价购买且不退货，说明产品价值匹配定价
- **穿着后的社交分享**：消费者主动拍照发布穿搭内容（不是品牌付费的），产品成为社交货币
- **品牌复访**：消费者在新一季主动回到品牌查看上新，表明品牌认知和信任已建立
- **尺码复购信心**：消费者不需要试穿就敢在线下单（"这个牌子的 M 码我穿着合适"）
- **推荐时描述品牌特征**：消费者能清晰说出品牌的风格定位和差异点

### 弱 PMF 信号（需更多数据验证）

- 社交媒体种草量高但转化低（"好看"不等于"会买"）
- 单一爆款销量高但其他款式平平（产品力 ≠ 品牌力）
- 线上流量高但退货率也高（尺码/面料/上身效果不及预期）

### 伪 PMF 信号警告

- **低价冲量的高销量**：9.9 包邮的销量不代表品牌有 PMF，只是价格驱动的无差异化消费
- **单一爆款的一次性热度**：一个款火了不代表品牌成立，下一季能否持续是关键
- **KOL 带货的瞬时脉冲**：主播粉丝的冲动购买，退货率常达 30%-50%
- **促销季的集中爆发**：双 11/618 的销量有 50%+ 是价格驱动，不代表正价购买意愿
- **跟风热度≠品牌认知**：消费者买的是"这个趋势"而非"这个品牌"

## 4. OmniscientAgent 指引

### INIT 阶段行业特殊性

OmniscientAgent 在处理时尚零售案例时，必须在 INIT 阶段锚定以下行业基线：
- **品牌基础死亡率 70%+**（3 年内）：默认假设品牌大概率无法存活到第三年
- **库存周转是生命线**：周转天数 > 90 天是危险信号，> 120 天是死亡信号
- **季节窗口不可逆**：错过当季销售窗口的库存不会"下季再卖"，而是持续贬值
- **爆款 ≠ 品牌**：一次性的热度不构成 PMF 证据，必须观察品牌维度的持续信号
- **退货率基线 20%-30%**：线上服装退货率普遍在 20%-30%，高于此值是产品问题

### Agent 配置建议

- Star Agent 配置为：时尚博主/穿搭 KOL、买手店买手、品牌主理人/设计师、时尚媒体编辑
- Sea Agent 群体结构：**10% 潮流引领者**（关注时尚趋势，愿意为设计溢价买单）→ **30% 趋势追随者**（看到流行才跟进，注重性价比）→ **60% 刚需购买者**（按需求购买，不追趋势，价格和实用性优先）
- 刚需购买者群体对品牌和趋势不敏感——他们的需求是"合适、耐穿、不贵"

### 涌现检测规则

- **正向涌现**：趋势追随者群体自发讨论品牌（而非单一产品），形成"这个牌子不错"的共识——品牌 PMF 的关键信号
- **负向涌现**：潮流引领者试穿/试用后沉默（不分享不推荐）——产品连最易感人群都没打动
- **伪涌现**：围绕单一爆款的集中讨论——可能是产品级热度而非品牌级 PMF，需观察其他款式是否连带增长

## 5. Star/Sea 行为锚点

### Star Agent（KOL/行业关键人物）典型行为

- **时尚博主/穿搭 KOL**：关注产品是否有"出片价值"——穿上身好不好看、能不能做出差异化的穿搭内容、粉丝会不会问链接。如果产品没有视觉冲击力或话题点，KOL 不会主动推荐
- **买手店买手**：以商业眼光审视——这个品牌的客单价、毛利空间、目标客群是否匹配自己的渠道。关注"下一季还能不能卖"而非"这一季好不好看"
- **品牌主理人/设计师**：以竞品视角评估——设计语言是否有辨识度、供应链能力是否支撑品牌扩张、定价策略是否可持续

### Sea Agent（消费者群体）典型反应

- **潮流引领者（10%）**：关注品牌调性、设计独特性、是否"小众但有品味"。他们追求差异化，对大众品牌天然抵触。正面反馈表现为主动分享穿搭、标注品牌名称、在评论区当"自来水"
- **趋势追随者（30%）**：关注"现在流行什么"，社交媒体热度是购买决策的重要参考。他们需要看到足够多的人在穿/推荐才会行动，价格在可接受范围内即可。复购取决于品牌是否持续"在潮流上"
- **刚需购买者（60%）**：以功能需求驱动——换季需要新衣服、旧衣服破了需要替换、特定场合需要特定服装。他们对品牌无忠诚度，购买标准是"合适+不贵"。这个群体对品牌 PMF 的贡献有限，但对销量基本盘有支撑作用

### "沉默多数"表现

时尚零售的沉默多数是那 60% 的刚需购买者。他们对品牌动态几乎无感知，不关注上新、不浏览穿搭内容、不参与社交讨论。他们的购买行为高度场景化（换季、出行、社交场合），与品牌的关系是"碰巧买到"而非"主动选择"。在模拟中，如果刚需购买者群体没有反应，这是符合预期的——品牌 PMF 的验证对象应聚焦在前 40%（引领者 + 追随者）。

## 6. 专属评分维度

### 社交货币价值（Social Currency Value）

> 参见通用评分卡（`rubrics/scorecard-dimensions.md` → 消费品扩展维度 → 社交货币价值）。
> 在时尚行业中，社交货币价值是品牌能否建立溢价的核心指标。"被看到穿"比"穿着舒服"在品牌建设中更重要。

### 品牌溢价接受度（Brand Premium Acceptance）

方向：1 = 只认价格，5 = 品牌即购买理由

> 时尚零售的利润空间取决于品牌能否支撑溢价。没有品牌溢价的服装生意本质上是
> 价格战——而价格战在这个行业意味着库存积压和资金链断裂。

| 分数 | 行为锚点 |
|------|----------|
| 1 | 只接受最低价位，品牌不是考量因素。消费者的购买标准纯粹是"最便宜+能穿"，在拼多多和 1688 比价是常态。品牌标签对购买决策没有任何影响。 |
| 2 | 愿意为品牌付出 10%-20% 溢价，超过即放弃。消费者认可品牌但价格仍是首要考量，"这个牌子是不错但太贵了还是算了"。会等到打折才入手。 |
| 3 | 接受品牌定价但会比较性价比，"贵一点可以但要值"。消费者认可品牌价值，但仍然会与同档位品牌比较，需要说服自己"这个价格买到了什么"。不会无条件接受价格。 |
| 4 | 认可品牌价值，价格在合理范围内不是首要考量。消费者因为品牌的风格、品质、调性而购买，价格在心理预期范围内即可接受。"这个牌子的东西值这个价"。 |
| 5 | 愿意为品牌付出 2x+ 溢价，品牌本身即是购买理由。消费者视品牌为个人风格/身份的表达，"就是要穿这个牌子"。品牌有明确的文化认同和情感连接，消费者对价格脱敏。 |

### 季节适配性（Seasonal Fit）

方向：1 = 严重错位，5 = 完美匹配且引领趋势

> 时尚零售的时间窗口极为残酷。产品与季节的匹配度直接决定库存周转效率——
> 错过窗口的产品不是"卖慢一点"，而是"变成库存负担"。

| 分数 | 行为锚点 |
|------|----------|
| 1 | 与当前季节/趋势完全无关或严重错位。夏天推冬季款、流行简约时推浮夸设计。产品在当前市场语境中找不到需求场景，库存消化将极为困难。 |
| 2 | 与季节有一定关联但时机偏早或偏晚。产品方向大致正确但上市节奏失误——过早则市场需求尚未激活，过晚则消费者已完成购买。需要大量营销投入才能创造需求。 |
| 3 | 在合理的季节窗口内，但不是最佳时机。产品与季节需求基本匹配，消费者能找到使用场景，但没有踩中需求高峰期。销售可行但不会自然爆发。 |
| 4 | 精准踩中季节需求，产品与时令趋势契合。上市时机恰好在需求启动期，产品风格/功能与当季消费者偏好高度一致。自然流量转化率高，不需要过度营销驱动。 |
| 5 | 完美匹配季节窗口且引领趋势。产品不仅满足当季需求，还定义了当季的审美方向。消费者和 KOL 自发将其视为"本季必入"，形成品类引领效应。极为罕见。 |

## 7. 常见陷阱与反模式

### 陷阱 1：把爆款当品牌 PMF

一个款式的爆火不等于品牌成立。大量"网红品牌"靠单一爆款冲到前排，但第二季就销声匿迹——因为消费者买的是"那个款"而非"那个品牌"。**LLM 乐观偏误**：LLM 容易被单一产品的高人气数据打动，生成"品牌已获得市场认可"的过度乐观判断。必须追问：其他款式是否连带增长？消费者能否说出品牌名字？

### 陷阱 2：忽视库存周转的致命性

库存周转天数 > 90 天是死亡信号。时尚品的价值随时间急速衰减——本季的热门款下季就是"过时款"，打折清仓还要占用现金流和仓储。**LLM 乐观偏误**：LLM 不会主动考虑库存管理这一"后端问题"，但它是时尚零售第一大死因。品牌可能产品很好、口碑很好，但因为库存管理失败而倒闭。

### 陷阱 3：低价冲量误判为市场需求

9.9 包邮 T 恤的百万销量不代表品牌 PMF——这只是价格驱动的无差异化消费。消费者对品牌毫无认知和忠诚度，随时会被更低价的替代品分流。**LLM 乐观偏误**：LLM 容易将"高销量"等同于"强需求"，忽略价格因素的扭曲效应。

### 陷阱 4：误读社交媒体热度

小红书上"好好看啊"的评论和收藏数不等于购买转化。时尚内容的互动率天然高（视觉吸引力强），但从"好看"到"值得买"到"真的买了"的漏斗极为陡峭。种草到拔草的转化率通常只有 2%-5%。**LLM 乐观偏误**：LLM 会被社交媒体上的高互动数据误导，高估产品的实际商业价值。

### 陷阱 5：忽视线下体验的不可替代性

尽管线上占比已超 50%，但线下试穿体验对中高端品牌仍然至关重要。退货率差异是证据——纯线上品牌的退货率（25%-40%）远高于有线下渠道的品牌（10%-20%）。一个"线上很火但退货率 40%"的品牌并不比"线上平淡但退货率 10%"的品牌更有 PMF。
