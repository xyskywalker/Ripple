# PMF 等级判定 Rubric

> 等级由 OmniscientAgent 在 SYNTHESIZE 阶段基于所有证据综合判断。
> Sean Ellis 对标值仅作为认知锚点，不是通过数字直接计算。
> 每个等级旁必须附注责任声明。

## 等级定义

### A 级 — 强 PMF

**含义**：目标用户群有强烈需求共鸣和传播动力。

**认知锚点**：对标 Sean Ellis ≥ 40%（"非常失望"比例）

**判定条件**（需同时满足）：
- 传播层：Star/Sea Agent 均表现出强烈需求共鸣信号，传播能量持续而非脉冲式
- 合议层：Tribunal 评审多数维度评分 ≥ 4，无维度评分 ≤ 2
- 一致性：传播层与合议层结论方向一致

**行为特征**：
- 用户说"这正是我需要的"并且会主动告诉别人
- 传播不需要推动，用户自发成为传播节点
- 竞品存在但用户有明确的选择理由
- 采纳路径清晰，障碍在可接受范围内

### B 级 — 有潜力

**含义**：特定细分群体有共鸣，需要聚焦或调整。

**认知锚点**：对标 Sean Ellis 25-39%

**判定条件**（满足其一）：
- 传播层显示明显的群体分化——某些群体反应强烈而另一些无感
- 合议层多数维度评分 3-4，有 1-2 个维度突出（≥4）
- 整体信号积极但有明确的短板需要改进

**行为特征**：
- 特定群体会说"这个对我有用"但不会向所有人推荐
- 传播发生在特定圈层内，跨圈层时能量衰减明显
- 产品有差异化但不足以让所有目标用户兴奋
- 存在可解决的采纳障碍或价值传递不够清晰

### C 级 — 弱信号

**含义**：有一定兴趣但不强烈，需要显著调整。

**认知锚点**：对标 Sean Ellis 15-24%

**判定条件**（满足其一）：
- 传播层能量快速衰减，Agent 反应以"有点意思"为主
- 合议层多数维度评分 2-3，没有突出维度
- 需求存在但产品解决方案不够有吸引力

**行为特征**：
- 用户听了会说"嗯，还行"但不会采取行动
- 几乎没有自发传播，信息止于首次接触者
- 竞品优势更明显，用户找不到切换的理由
- 采纳障碍相对于产品提供的价值而言偏高

### D 级 — PMF 不足

**含义**：产品与目标市场不匹配。

**认知锚点**：对标 Sean Ellis < 15%

**判定条件**（满足其一）：
- 传播层 Agent 反应以忽略或无感为主
- 合议层多数维度评分 ≤ 2
- 产品解决的不是目标市场的核心问题

**行为特征**：
- 用户的反应是"跟我有什么关系？"
- 没有传播动力，信息不产生任何涟漪
- 没有清晰的差异化，或差异化对用户无意义
- 即使免费，用户也不想花时间尝试

### F 级 — 高风险

**含义**：存在根本性问题或市场障碍。

**认知锚点**：模拟中发现致命问题

**判定条件**（满足其一）：
- 传播层出现负面涌现（如负面口碑快速传播、群体抵制信号）
- 合议层 Devil's Advocate 发现无法解决的结构性风险
- 产品触犯了目标市场的核心禁忌或底线

**行为特征**：
- 用户的反应是负面的（不仅是无感，而是反感）
- 出现主动抵制或负面传播信号
- 存在法律、合规、伦理等根本性障碍
- 市场时机严重错位（如已有免费替代品完全覆盖需求）

---

## 等级判定指导原则

1. **综合判断，不套公式**：等级由 OmniscientAgent 综合传播层和合议层的全部证据做出判断，不是简单的分数加权。

2. **传播层与合议层需一致**：
   - 两层结论一致 → 置信度高（high confidence）
   - 两层轻微分歧 → 置信度中（medium confidence），报告中说明分歧原因
   - 两层严重矛盾 → 置信度低（low confidence），等级取保守值

3. **疑点利益归保守**：当证据不充分或结论不明确时，倾向于给更保守的等级。宁可低估不可高估。

4. **责任声明**：每个等级判定旁必须附注——"本等级基于 Agent 模拟，仅供参考。真实 PMF 验证需要市场数据支撑。"

5. **不给 A+ 或 F-**：等级范围严格为 A/B/C/D/F，不存在加减号变体。
