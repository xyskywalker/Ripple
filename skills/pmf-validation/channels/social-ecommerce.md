# 社交电商渠道画像

> 代表平台：微信小程序商城、拼多多拼团、社区团购（美团优选、多多买菜）
> 核心机制：关系链传播 → 信任背书 → 社交裂变

## 渠道概述

社交电商是中国市场最具特色的电商形态，其核心驱动力不是算法也不是搜索，而是人际关系链。用户的购买决策深度嵌入社交网络——拼团需要邀请好友、砍价依赖社交助力、社区团购通过邻里信任传播。这种模式将商品交易与社交关系绑定，使得传播路径天然沿着真实的社交图谱展开。

社交电商的传播动力学具有鲜明的"裂变-疲劳"双面性。一方面，基于信任关系的推荐转化率远高于算法推荐或广告投放；另一方面，社交关系被商业化过度消耗后会快速产生疲劳效应——用户开始屏蔽拼团消息、忽略群内链接、甚至对频繁分享的好友产生反感。这种疲劳效应是社交电商渠道中最容易被忽视的负反馈机制。

对PMF验证而言，社交电商渠道的核心挑战在于区分"利益驱动的分享"和"价值驱动的推荐"。拼多多式的补贴裂变可以在短时间内制造惊人的传播数据，但当补贴消失后传播是否持续才是PMF的真正检验。免费获客与付费留存之间的鸿沟，是社交电商中最常见的PMF幻觉来源。

## 传播动力学

### 信息流转模式

社交电商的传播遵循"关系链裂变"模型：

1. **种子触发**：初始用户通过分享链接、拼团邀请等方式在社交网络中发起传播
2. **信任放大**：基于真实社交关系的推荐具有天然信任背书，转化效率高于陌生推荐
3. **裂变扩散**：每个新参与者成为下一轮传播的节点，形成指数级扩散潜力
4. **疲劳衰减**：社交关系被商业化消耗后产生疲劳，传播速率快速下降

### 基准参数

| 参数 | 参考值 | 说明 |
|------|--------|------|
| Wave 时间窗口 | 4-8h | 社交传播集中在活跃时段，拼团有限时机制加速决策 |
| 每轮衰减率 | 15-20% | 关系链传播有衰减但比算法推荐持久 |
| 跨群体传播系数 | 0.20-0.30 | 社交关系提供跨圈层桥梁，但受限于弱连接密度 |

### CES 类行为权重参考

社交电商中行为的信号意义与社交成本高度相关：

- **分享**（权重约6）：愿意在社交关系中转发，意味着以个人信用做背书
- **助力/拼团**（权重约8）：不仅自己参与还动员他人，社交行动成本高
- **购买**（权重约5）：在社交电商中购买行为的信号强度需折扣——低价拼团购买可能仅因价格而非产品
- **复购**（权重约7）：在没有新的社交裂变触发下主动回购，表明产品本身有价值
- **推荐好友**（权重约10）：非利益驱动地主动推荐给他人，是最强的PMF信号

## PMF 信号特征

### 强PMF信号

- **非补贴驱动的自发分享**：在没有砍价、拼团优惠等利益激励的情况下，用户仍然主动在社交网络中推荐产品
- **老用户主动拉新**：非利益驱动的自发推荐行为，用户因为产品体验好而非获得奖励来推荐
- **社群内自发讨论产品体验**：微信群、朋友圈中出现非广告性质的真实使用分享和讨论
- **拼团成团率高且客单价非低价**：中高客单价商品仍能顺利成团，说明需求真实而非价格驱动
- **社区团购中团长主动推荐**：团长基于用户反馈而非佣金主动在群内推荐

### 弱PMF信号（警惕误判）

- 补贴期间的高裂变系数：免费或极低价格驱动的传播不代表付费意愿
- 拼团量大但复购率低：用户可能只是凑单帮忙，并非真实需求
- 社群活跃度高但转化低：群内互动可能是聊天社交而非购买意向
- 砍价助力人数多但实际购买者少：大部分助力用户只是帮忙，没有自购意愿

## 关键指标锚点

以下为真实市场中的基线水平，模拟中应以此为锚：

| 指标 | 基线范围 | 反乐观锚点 |
|------|----------|------------|
| 注意力捕获率（Attention） | 10-20% | 社交推荐的关注度高于广告，但微信分享打开率不足5% |
| 行动转化率（Action） | 3-6% | 基于信任推荐的转化率相对高，但绝对值仍然有限 |
| 推荐传播率（Recommend） | 2-5% | 社交电商的推荐率高于其他渠道，但仅限于强关系链 |
| 拼团成团率 | 差异极大 | 刚需高频品>80%，非刚需品<30%，不可一概而论 |
| 微信分享打开率 | <5% | 大量分享链接被忽略，尤其是被频繁骚扰的用户 |
| 补贴停止后留存率 | 10-30% | 免费/低价获得的用户，恢复正常价格后流失严重 |
| 社交疲劳周期 | 2-4周 | 同一用户对同一产品/品牌的社交传播耐受度有限 |

**核心反乐观认知**：社交电商中最大的PMF幻觉是将"补贴裂变"误判为"产品需求"。拼多多模式证明了社交裂变的强大传播力，但也揭示了一个残酷现实——当补贴消失后，大量用户随之流失。真正的PMF检验是在去除所有利益激励后，用户是否仍然愿意为产品付费并主动推荐。社交关系是一种可消耗资源，过度商业化会导致关系链的信任折损，这种损耗一旦发生极难恢复。
