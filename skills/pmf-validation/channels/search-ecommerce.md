# 搜索电商渠道画像

> 代表平台：天猫、京东、亚马逊中国
> 核心机制：主动搜索 → 评价级联 → 理性比较

## 渠道概述

搜索电商是中国电商生态的基础设施层。其核心逻辑是"人找货"——用户带着明确或半明确的购买意图主动搜索，在搜索结果中浏览、比较、决策。与算法推荐电商的冲动消费形成鲜明对比，搜索电商中的用户行为链路更长、决策更理性、价格敏感度更高。

搜索电商的传播动力学以"评价级联"为核心特征。一个商品的评价体系（好评率、评价内容、追评、问答）构成了一个缓慢但持久的信息传播场。新用户的购买决策高度依赖已有评价，而每一个新评价又成为后续用户的决策依据，形成正向或负向的级联效应。这种传播速度远慢于算法推荐，但具有显著的长尾特性——一个建立了强评价体系的商品可以持续数月甚至数年获得稳定流量。

对于PMF验证而言，搜索电商的关键挑战在于"从零到一"的冷启动。新品牌在搜索电商中面临零搜索量、零评价、零排名的三重困境。品牌词搜索量的自然增长是最可靠的PMF信号之一，因为它代表用户在其他渠道被种草后主动来搜索电商验证和购买——这是跨渠道PMF验证的典型路径。

## 传播动力学

### 信息流转模式

搜索电商的传播遵循"搜索-评价-级联"模型：

1. **搜索触发**：用户输入关键词，平台返回排序结果（权重因子：销量、评分、价格、广告出价）
2. **详情浏览**：用户进入商品详情页，核心关注点为价格、评价、商品参数
3. **评价级联**：已有评价影响新用户决策，新评价又反馈进评价体系
4. **竞品比较**：用户在多个同类商品间反复比较，决策周期较长

### 基准参数

| 参数 | 参考值 | 说明 |
|------|--------|------|
| Wave 时间窗口 | 24-48h | 用户决策周期长，搜索→比较→下单需要时间 |
| 每轮衰减率 | 10-15% | 搜索流量衰减慢，优质商品可长期维持排名 |
| 跨群体传播系数 | 0.10-0.15 | 搜索行为高度个人化，跨群体传播弱 |

### CES 类行为权重参考

搜索电商中的用户行为信号更为直接且与购买意图强关联：

- **搜索点击**（权重约1）：进入详情页仅表示初步兴趣，搜索结果页CTR通常5-15%
- **收藏**（权重约2）：列入候选清单，表示有一定购买意向但尚在比较
- **加购**（权重约3）：比收藏更强的购买信号，但加购放弃率极高
- **购买**（权重约8）：直接的付费行为
- **好评**（权重约5）：正向评价对后续用户的级联影响
- **追评**（权重约7）：使用一段时间后仍愿意追评，是产品持续满意度的强信号

## PMF 信号特征

### 强PMF信号

- **品牌词搜索量持续增长**：用户从其他渠道了解产品后主动来搜索电商搜索品牌名，表明品牌认知正在建立
- **非广告位自然排名上升**：在未投放搜索广告的情况下，自然搜索排名持续提升
- **评价中出现竞品对比选择**：用户在评价中主动提到"对比了XX后选择了这个"，是极强的产品差异化信号
- **追评质量高**：追评中描述具体使用场景和持续满意度，而非敷衍了事
- **类目关键词搜索下的自然转化**：在通用类目词（非品牌词）搜索中也能获得转化

### 弱PMF信号（警惕误判）

- 通过刷单和返现获得的好评数量：虚假评价不代表真实满意度
- 广告位带来的高流量高销量：付费搜索广告的转化不等同于自然需求
- 大促期间的销量峰值：价格折扣驱动的销量不代表正价下的PMF
- 加购量高但转化率低：可能是价格犹豫或同类竞品太多

## 关键指标锚点

以下为真实市场中的基线水平，模拟中应以此为锚：

| 指标 | 基线范围 | 反乐观锚点 |
|------|----------|------------|
| 注意力捕获率（Attention） | 15-30% | 搜索用户意图明确，详情页到达率相对高，但仍有大量用户只看搜索结果不点击 |
| 行动转化率（Action） | 3-8% | 搜索后90%以上用户不会购买，加购后放弃率70%以上 |
| 推荐传播率（Recommend） | 0.3-1% | 搜索电商用户极少主动推荐，评价是主要的间接推荐形式 |
| 加购放弃率 | 70%+ | 大部分加购行为不会转化为实际购买 |
| 好评真实度 | 需折扣估算 | 刷单、返现好评比例因类目差异大，部分类目虚假好评占30-50% |
| 新品牌搜索量 | 接近零 | 新品牌在搜索电商的初始搜索量几乎为零，需要外部渠道引流建立认知 |

**核心反乐观认知**：搜索电商中"有流量"和"有需求"是两件完全不同的事。通过SEO优化、广告投放获得的流量是平台分配的，不代表市场自发需求。真正的PMF验证要观察品牌词搜索量的自然增长趋势、非促销期的稳定转化率、以及用户评价中是否出现"主动选择本品而非竞品"的真实表达。好评数量本身不可靠——在中国电商生态中，好评的真实度需要大幅折扣估算。
