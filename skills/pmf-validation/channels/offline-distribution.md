# 线下流通零售渠道画像

> 适用于以货架分销为核心的快消零售场景：商超/大卖场（沃尔玛、永辉、大润发）、便利店（全家、罗森、美宜佳、7-Eleven）、社区店、夫妻杂货店等。
> 传播逻辑为"货架可见性 → 即时决策 → 复购惯性"，渠道铺货率和陈列位置是生死线。

## 渠道概述

线下流通零售是中国消费品最庞大也最残酷的战场。这个渠道的核心逻辑不是"体验"而是"便利性"和"惯性"——消费者在货架前的决策时间通常只有 3-7 秒，品牌认知、包装辨识度和价格锚点在这几秒内完成全部博弈。不同于体验零售的"深度说服"，流通零售拼的是"条件反射式购买"。

这个渠道对新品牌极不友好。消费者的品牌切换意愿极低——研究表明 80%+ 的快消购买行为是惯性驱动的，消费者进入货架区域时心里已经有了默认选择。新品牌要打破惯性，要么靠极端的价格优势，要么靠高频次的广告轰炸建立认知，两者的成本都极高。更残酷的是，即便成功上架，商超的新品 3 个月存活率不到 30%——动销不达标就会被清退。

渠道成本结构是另一个被低估的障碍。商超的进场费、条码费、堆头费、促销扣点、账期占用构成了一个复杂的成本矩阵。便利店虽然进场门槛较低，但 SKU 数量极为有限（一个品类通常只容纳 3-5 个品牌），淘汰率极高。新品牌必须认识到：上架只是开始，活下来才是真正的考验。

## 传播动力学

### 信息流转模式

流通零售的传播模式是"被动发现 + 惯性锁定"：

1. **货架曝光**：消费者在购物过程中被动注意到产品（高度依赖陈列位置）
2. **即时决策**：3-7 秒内完成"拿起-查看-放回/放入购物车"的决策
3. **试错性购买**：首次购买往往是低风险试错，期望值不高
4. **复购锁定**：如果产品满足需求，进入惯性购买循环，品牌切换成本上升
5. **极弱口碑传播**：快消品的口碑传播意愿极低，除非产品远超预期

### 基准参数

| 参数 | 数值 | 说明 |
|------|------|------|
| Wave 时间窗口 | 3-7 天 | 与购物频率挂钩（商超周频、便利店日频） |
| 每轮衰减率 | 10-15% | 无持续曝光支撑则快速消退 |
| 跨群体传播系数 | 0.05-0.10 | 快消品口碑传播极弱，几乎无跨群体效应 |

### CES 行为权重

| 行为节点 | 权重 | 备注 |
|-----------|------|------|
| 货架注意 | 1 | 被动注意，信号极弱 |
| 拿起查看 | 3 | 产生了主动兴趣，已超越多数竞品 |
| 首次购买 | 5 | 试错性购买，不等于认可 |
| 复购 | 8 | 强信号——消费者用行为投票 |
| 推荐给他人 | 10 | 快消品推荐极罕见，一旦出现是超强信号 |

## PMF 信号特征

### 强 PMF 信号

- **无促销时自然动销**——排除折扣、买赠等促销因素后仍有稳定的售出量
- **消费者指名购买**——到店后直接寻找该品牌，不接受替代品推荐
- **便利店/商超主动要求补货**——渠道方从"被铺货"转变为"主动要货"
- **货架位置不佳仍能动销**——不依赖黄金位陈列也能卖出去
- **竞品促销期间仍保持销量**——消费者不被竞品降价吸引走

### 弱 PMF 信号 / 噪音

- 促销期间的销量飙升（价格驱动≠产品认可，促销停则销量停）
- 铺货率高但动销率低（上了架但卖不动，渠道成本在失血）
- 仅在特定区域/特定渠道有销量（可能是导购推荐或特殊陈列位效应）
- 试吃/试用活动期间的数据（一次性尝试，不代表持续需求）

## 关键指标锚点

| 指标 | 现实基准 | 说明 |
|------|----------|------|
| 品类购物者→货架注意 | 10-25% | 多数消费者直奔惯性品牌，不看新品 |
| 注意→首次购买 | 3-8% | 快消品的试错购买门槛看似低，但真正行动的人很少 |
| 首购→复购率（季度） | 15-25% | 快消品复购率，扣除促销驱动 |
| 购买→推荐他人 | 0.2-1% | 快消品口碑传播意愿极低 |
| 新品 3 个月存活率 | <30% | 商超/便利店新品淘汰率极高 |

### 反乐观偏差锚点

- **80%+ 消费者按惯性购买**——品牌切换意愿极低，新品突围难度远超预期
- **货架位置 = 生死线**——非黄金位陈列几乎无人问津，但黄金位需要高额费用
- **新品上架 3 个月内存活率 <30%**——大多数新品在动销不达标后被清退
- **商超进场费 + 条码费是重大成本障碍**——中小品牌可能还没验证 PMF 就因渠道成本耗尽资金
- **便利店 SKU 极有限**——每个品类通常只有 3-5 个位置，淘汰率极高
- **促销带来的销量不是 PMF 证据**——停止促销后的自然动销才是真实信号
